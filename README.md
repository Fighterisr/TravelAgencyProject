# TravelAgencyProject

## Visit the website at:
http://travelagencyproject.herokuapp.com/

## Requirements:
Download and install Node.js

## Instructions:
1. Clone the project.
2. Open terminal and run the command: "npm install".
   Npm will read dependencies and install everything automatically.
3. Run the command: "npm run start".
4. Open your browser and enter the address: 127.0.0.1:3000
